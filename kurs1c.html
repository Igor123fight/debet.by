Содержание курса 1С для ИП

Здесь представлено все содержание курса по работе с 1С для ИП

1.Знакомство с программой (1-5, -3). Префикс для организации

2.Знакомство с типами документов, отчетов и прочего в 1С

3.Создание документа поступления (+справочники)

4.Отчеты руководителю (остатки товаров), отчет по МОЛ, ОСВ

5.Создание документа реализации Подбор РО.ВУ (стандартные печатные формы)

6. Отчеты руководителю (остатки товаров), отчет по МОЛ, ОСВ

7. Формирование книги (товаров, КУДИР)

8. Поступление услуг (отдельная папка счет 76.7)

9. ОСВ, формирование книги (КУДИР)

10. Реализация услуги, ОСВ, формирование книги (КУДИР)

11. Поступление материалов

12. Услуга + материалы, ОСВ, формирование книги (материалов, КУДИР)

13. Отчет о розничных продажах (ОПК (виды оплат)), ОСВ,книги – ошибка, после исправления добавлю

14. Вовзраты товаров (поставщику, от покупателя) отражение в книгах

15. Производство (котловой/позаказный, номенклатурная группа, статья затрат, ПЛАНОВАЯ ЦЕНА), требование-накладная/ОПЗ, ОСВ, Отчет по МОЛ, Книга товаров и книга материалов

16. Закрытие месяца (смотрим как закрылось производство)

17. Реализация ГП, осв, книги

18. Списание материалов на собственные нужды (книги)

19. Взаиморасчеты с контрагентами (ОСВ, Акты сверки, отчеты руковод, сроки долга)

20. Обмен с банком, создание документов вручную (виды операций), проверка загрузки выписки, ОСВ, книги (по оплате)

21. Касса (РКО, ПКО, виды), отчет о розничных продажах (как проходит продажа по банку и кассе), печать кассовой книги

22. Зарплата: фл, сотрудники, прием на работу, льготы по ПН, производственный календарь, график работы (посменно, 5/2, на 0,5, почасовая), табель, начисление зп, удержание, больничный, отпуск, увольнение, комп при увольнении, 4-фонд, пу-2, пу-3, НАСТРОЙКА ПО З/П

23. Отражение зарплаты в КУДИР, редактирование КУДИР (включаются расходы по з/п, ФСЗН и БГС)

24. Авансовые отчеты для ИП (в каких случаях, альтернативная схема по товарам/материалам) (добавить пример по топливу)

25. Отражение НДС в проводках входящий(неплательщик НДС, поступление из россии, заявление). Создание ЭСЧФ и загрузка с портала (галки, СТАТУС). Проверка ЭСЧФ на портале и в 1с.

26. ЭСЧФ по авансовым отчетам, эсчф по возврату поставщику

27. Отражение НДС исход. в проводке. Создание ЭСЧФ исходящих (реализация (учет по оплате), нумерация, отправка. НДС 60 дней. С 1С и в черновики (выгрузка в ХМЛ)

28. Формирование Книги НДС, заполнение декларации, вычет НДС. ОСВ ( в пределах КО 68.2.1)

29. Как создать документ на основании ЭСЧФ, загрузка из эксель (рассмотрим НТТ, РАССКАЗАТЬ АТТ)

30. Товарный отчет, ОСВ

31. Отражение оплаты (ОПК, ПКО), загрузка СККО

32. Перемещение в опт, реализация юрикам

33. Формирование книги суммового учета

34. ЭСЧФ розничный

35. КУДИР, Книга НДС

36. Работа с валютой (курсы валют, как загрузить вручную, как обозначить валютный документ, переоценка, настройка переоценки, расчеты в у.е – запрет)

37. Загрузка валютной выписки

38. Покупка с перечислением, продажа с перечислением

39. Покупка / продажи валюты, конверсия

40. Реализация услуг в валюте (по какому курсу если по отгрузке/по оплате)

смотрим книгу (реализация учет по оплате – КУДИР).

41. Переоценка в книге (как убрать)
